# Management プロジェクト

このプロジェクトは、開発者または運用担当者向けの管理用の機能を提供するコマンドラインアプリケーションです。
単なるシェルスクリプトだけでは実現が難しい複雑な処理や、 NijoAppScaffold が生成するC#コードと連携する必要のある処理を実装するために使用されます。

このプロジェクトは Nijo のデモプロジェクトに同梱されていますが、必ずしも利用は必須ではなく、アプリケーションごとの要件に応じて別の手段で代替できるのであれば削除しても構いません。

## 機能例

* 自動生成されたC#コードを参照し、データベース定義更新用SQLの一部を生成する
* デバッグプロセスを立ち上げる
  * Nijo のデモプロジェクトは `npm run dev` と `dotnet run` を同時に実行する必要がある。
  * ホットリロードなどもサポートしようとすると、単純なシェルスクリプトでは実現が難しい。
* デプロイ手順の一部を自動化する
  * デモでは未実装
  * クラウドが提供する機能などを駆使して代替できるのであれば不要

## どのようなコマンドが使えるかの確認方法

ビルドされたexeを引数なしで実行するか、 `--help` を付けて実行すると、利用可能なコマンドの一覧が表示されます。

## コマンド追加時手順

`System.CommandLine` ライブラリを使用しています。

1. `Program.cs` にコマンドを追加
2. `Commands` フォルダ内にコマンドの実装クラスの追加
