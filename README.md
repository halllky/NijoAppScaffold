# Nijo App Scaffold

**複雑な業務を、目に見える形に。スキーマ駆動アプリケーション生成フレームワーク**

Nijoは、複雑な業務フローを持つエンタープライズアプリケーションを効率的に開発するための、フルスタック・コード生成フレームワークです。
データ構造を定義した1個のXMLファイルを中心とし、堅牢なアプリケーションの土台を自動生成します。

## 特徴

- **可視化**: 専用GUIエディタでデータ構造をグラフ化し、チームで共有・合意形成が可能。
- **拡張性**: 自動生成コードと手動実装が分離されており、自由にロジックを拡張可能。
- **型安全**: DBからバックエンド、フロントエンドまで型定義を一気通貫で自動生成。

## クイックスタート

### インストール
Releases ページから最新のバイナリをダウンロードし、PATHを通してください。

### プロジェクト作成
```bash
nijo new MyProject
```
